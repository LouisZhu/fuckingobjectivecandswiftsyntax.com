How do I write in Objective-C and Swift?

Init a dictionary:

var dictionaryName = [keyType: valueType]()

Unwrap an optional value using guard:

guard let unwrappedValue = optionalValue else {
    dealWithNilValue()
}

continueUseUnwrappedValue()

Modularized constants:

struct ModuleName {
    static let constantA = constructConstantA()
    static let constantB = constructConstantB()
}

Enumeration with associated value:

enum EnumType {
    case BranchName(MyType, YourType)
}

...

switch enumValue {
case .BranchName(let myValue, let yourValue):
    dealWithMyValueAndYourValue()
}
